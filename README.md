Deep Rock Galactic Save Editor is a tool for editing save files of the game Deep Rock Galactic. It is written in Python and uses PyQt6 for the GUI. 
## Features
- Edit resources
- Edit season 4 progress
- Edit credits
- Edit perk points
- Edit dwarf level
- Unlock Overclocks
- Automatically backup
